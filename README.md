#####  Nombre del Proyecto: 
Manejo de Archivos de Texto y Validaciones en C#

### Descripción

Este proyecto trata sobre una aplicación de escritorio hecha en C# que permite registrar los datos de un empleado y guardarlos en un archivo de texto.
La aplicación valida que todos los campos estén completos y muestra mensajes de alerta para guiar al usuario durante el proceso de guardado, salida y apertura del archivo.

El archivo se guarda usando un cuadro de diálogo, parecido a cuando se guarda un documento de Word o PowerPoint, y los datos se muestran organizados en forma de tabla.

### Tecnologías Utilizadas

- Lenguaje: C#

- Framework: .NET (Windows Forms)

- Entorno de Desarrollo (IDE): Visual Studio

- Tipo de Aplicación: Escritorio
